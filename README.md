# Movie-App
Movie App is desktop application that uses Javascript/ES6 and TMDB (The Movie Database) Api.

# Below is a list of the methods and properties used

* querySelector
* createElement
* innerHTML
* appendChild
* insertBefore
* preventDefault
* dataset
* parentElement
* nextElementSibling
* classList
 * add
 * remove
* fetch
* bind
* setAttribute
* onClick
* toLowerCase
* target
